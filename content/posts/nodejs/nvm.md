## Install

```bash
yay -Sy nvm
```

## Install the latest version of Node

```bash
nvm install node
```

## Install a specific version of Node

```bash
nvm install 18.9.0
```

## Switch to a specific version of Node

```bash
nvm use 18.9.0
```
